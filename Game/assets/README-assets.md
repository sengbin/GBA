开源瓦片与像素素材来源（可商用/可重用资源优先）

- OpenGameArt: https://opengameart.org  （筛选 CC0/公共领域 或 明确允许商用的许可）
- Kenney Assets: https://kenney.nl/assets （多数为 CC0，适合快速原型）
- Itch.io 资源库: https://itch.io/game-assets （按许可筛选免费素材）
- Game-icons.net: https://game-icons.net （图标与 UI 元素，注意许可）
- GitHub 搜索: 搜索关键词 "tileset CC0" 或 "pixel art tileset" 可以找到许多合集

许可注意事项
- 优先使用 CC0（公共领域）以避免署名/衍生限制。
- CC-BY 需要保留作者署名。
- CC-BY-SA 会要求相同许可的衍生作品，请注意传播限制。

素材获取建议
- 原型阶段使用 Kenney 或 CC0 资源，开发成熟后替换为原创素材或获得明确授权。
