# GBA 开发指南

## 概述
本指南汇总了 GBA 开发所需的常用工具。

---

## 必备软件与下载链接
- Tiled（地图/瓦片编辑器）：https://mapeditor.org/
- devkitPro（GBA 工具链，含 devkitARM）：https://devkitpro.org/
  - Windows 快速入门：https://devkitpro.org/wiki/Getting_Started/Windows
- mGBA（模拟器）：https://mgba.io/downloads.html
- ImageMagick（图像缩放/处理）：https://imagemagick.org/script/download.php
- Python（用于运行转换脚本）：https://www.python.org/downloads/
  - Pillow（Python 图像库）：https://python-pillow.org/，安装：`pip install pillow`
- grit（GBA 图像/tiles 工具，可选）：https://www.coranac.com/projects/grit/
- Kenney（高质量免费游戏素材）：https://kenney.nl/assets ；itch.io 页面：https://kenney.itch.io/
- OpenGameArt（开源素材库）：https://opengameart.org/

> 说明：使用第三方素材前请查看素材许可（Kenney 通常为 CC0，但不同包可能不同）。
